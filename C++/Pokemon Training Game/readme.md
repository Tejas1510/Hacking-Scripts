# Pokemon Training Game
## Introduction
Suppose,You are a Pokemon trainer. Each Pokemon has its own power, described by a positive integer value. As you travel, you watch Pok√©mon and you catch each of them.
After each catch, you have to display maximum and minimum powers of Pokemon caught so far.
