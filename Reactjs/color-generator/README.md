# Color Generator

Users can generate color for their projects as their requirement.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Tejas1510/Hacking-Scripts
```

Go to the project directory

```bash
  cd Reactjs
  cd color-generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Tech Stack

**Client:** ReactJS
