# Python YouTube Downloader

A simple python GUI built using Tkinter an Pytube to facilitate video downloads from YouTube.

![Preview Image](preview.png)

## Libraries Used
* Python 3 <br/>
* Tinkter <br/>
* Pytube

## Dependency Installation

1. Install Pytube
```
$ pip install pytube 
```
