# JARVIS

## Introduction
Control windows programs with your voice.

## Requiremnts 
- python >= 3.7
- pyttsx3
- speech_recognition

## Install Requirments
> Download and install the latest version of [Python](https://www.python.org) or using pip.
> py install pyttsx3
> pip install speech_recognition

## Run it
> run the JARVIS.py file
